# Serene Industries Firmware

Official firmware releases for Serene Industries keyboards.

## Supported Keyboards

- **Icebreaker Hotswap** - QMK/VIA (.uf2)
- **Icebreaker Hall Effect** - QMK/VIA (.bin)
- **Icebreaker Wireless** - ZMK (.uf2)
- **Cleaver Hall Effect** - QMK/VIA (.bin)

## Download

Visit the [Releases](https://github.com/serene-industries/firmware/releases) page to download firmware for your keyboard.

## Flashing Instructions

1. Download the firmware file for your keyboard
2. Put your keyboard into bootloader mode using the key combination:
   - **Icebreaker Hotswap/HE/Cleaver**: WIN/CMD + LEFT SHIFT + ESCAPE
   - **Icebreaker Wireless**: FN + LEFT SHIFT + ` (TILDE)
3. A USB drive will appear on your computer
4. Drag and drop the firmware file onto the drive
5. The keyboard will automatically reboot with the new firmware
